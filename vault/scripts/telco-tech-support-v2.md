# Script: telco-tech-support-v2

Niet-bank scam — gericht op KPN-klanten. Twee fasen:
1. "Uw modem is overgenomen door een virus uit Rusland."
2. "Installeer alstublieft deze software zodat we mee kunnen kijken."

Eindigt met IBAN-vraag voor "abonnementsterugbetaling". Twee calls deze week, beide `[[scammer-voice-C9]]`.
